# docker nginx php normal
docker nginx php normal

## run 
````
  docker-compose up -d
````
## stop 
````
  docker-compose stop 
````
## restart
````
  docker-compose restart
````
## shutdown
````
  docker-compose down
````
## log
````
  docker-compose logs -ft
````

````
http://localhost:7000
http://localhost:7000/info.php
````
